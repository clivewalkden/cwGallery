This script is a basic gallery script that you supply the thumbnail and main images for and it displays the larger image when the thumbnail is clicked.

Basic Usage

The code below...

<div id="example1" class="clearfix">
    <div class="cw_image">
        <img src="http://farm9.staticflickr.com/8424/7720259766_ef2fc0c2bf.jpg">
    </div>
    <ul class="cw_thumbs">
        <li><a href="http://farm9.staticflickr.com/8424/7720259766_ef2fc0c2bf.jpg" class="cw_active"><img src="http://farm9.staticflickr.com/8424/7720259766_ef2fc0c2bf_q.jpg"></a></li>
        <li><a href="http://farm9.staticflickr.com/8440/7882885212_b2822e751e.jpg"><img src="http://farm9.staticflickr.com/8440/7882885212_b2822e751e_q.jpg"></a></li>
        <li class="last"><a href="http://farm8.staticflickr.com/7130/7882880940_56c68d4198.jpg"><img src="http://farm8.staticflickr.com/7130/7882880940_56c68d4198_q.jpg"></a></li>
    </ul>
</div>

<script>
// Finally call the script
$(function(){
    $('#example1').cwGallery();
});
</script>

Configuration Options

Property Type Default Description
thumb_container string cw_thumbs The class of the thumb container. Any <a> tags within this container are gleamed for their href which is the larger image
img_container string cw_image The container around the main image. The <img> tag within this is the main gallery target image.
active_class string cw_active The class applied when the thumbnail is selected. This class is added to the thumbnail when it is clicked to show which image is active.
animate_height boolean true If you want the height of the image to animate between tweens. If this is set to true then the height will animate when a different sized image is selected. If all your images are the same height then set this to false.
