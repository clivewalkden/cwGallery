CWGallery v0.1

This script is a basic gallery script that you supply the thumbnail and main images for and it displays the larger image when the thumbnail is clicked.

Basic usage

Configuration Options

The script has 3 options:

  • thumb_container
  • img_container
  • active_class
  • animate_height

thumb_container

The class of the thumb container.

Any <a> tags within this container are gleamed for their href which is the larger image

default: 'cw_thumbs'

img_container

The container around the main image

The <img> tag within this is the

default: 'cw_image'

active_class

The class applied when the thumbnail is selected.

This class is added to the thumbnail when it is clicked to show which image is active.

default: 'cw_count_warning'

animate_height

If you want the height of the image to animate between tweens.

If this is set to true then the height will animate when a different sized image is selected. If all your images are the same height then set this to false.

default: false
